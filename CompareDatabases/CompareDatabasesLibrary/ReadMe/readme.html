General Info

In DVD Profiler a profile is identified by either its UPC or EAN or DiscID and the locality of the profile.
If these two are the same a profile is considered identical.
This program finds identical profiles in two different databases and shows the differences in these two profiles.

This is needed when two different databases are merged into one, but a loss of information should be prevented.

It is also needed when the same database is managed on two different computers and there were changes made on both sides before synchronizing the database.
